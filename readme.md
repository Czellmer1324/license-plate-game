# License Plate Game

A full-stack web application that digitizes the classic road trip License Plate Game, allowing users to track and manage license plates spotted from different states.

This project demonstrates practical experience with full-stack development, containerization, and deployment of a live web application. The backend is built with Spring Boot and PostgreSQL. The API is secured using JWT authentication with Spring Security. The frontend is built with HTML, CSS, and JavaScript. The application is deployed on Render.

---

## Live Application

Access the deployed application here:

---

## Tech Stack

**Frontend:** HTML, CSS, JavaScript  
**Backend:** Java (Gradle), Spring Boot (Spring Security, Spring Data JPA), PostgreSQL  
**Deployment:** Docker, Render  
**Version Control:** Git & GitHub

---

## Features

- Track license plates spotted by state
- Interactive web interface
- JWT-based authentication and authorization
- CORS configuration for secure frontend-backend communication
- Containerized application setup
- Cloud-hosted deployment
